## Color & Theme Support

### Light Mode
- Primary button: Blue gradient
- Cards: White with subtle shadow
- Code blocks: Light gray background
- Success indicators: Green badges

### Dark Mode
- Primary button: Blue gradient (adjusted)
- Cards: Dark gray with subtle border
- Code blocks: Darker gray background
- Success indicators: Green badges (adjusted)

## Interactive Elements

### Buttons
- **Scan Now** (Dashboard CTA): Large, primary, gradient
- **Back to Dashboard**: Outlined, secondary
- **Clear History**: Outlined, destructive (red text)
- **Try Again**: Primary (on error state)

### Badges
- **Scanning...**: Blue, semi-transparent
- **JSON**: Outlined, small (in table)

### Tables
- Hover effect on rows
- Responsive column widths
- Truncated text with ellipsis

### Collapsible
- Arrow icon rotates on expand
- Smooth transition
- Muted background when collapsed
